# Markdown – Cheat Sheet



## Markdown Syntax – Übersicht



### Heading (Titel)

```md

# H1

## H2

### H3



Um nur einen Textteil fett zu machen:
**bold text**



Um Ihn kursiv zu machen:


*italicized text*



Blockzitat:
> blockquote



Eine geordnete Liste zu machen:
1. First item

2. Second item

3. Third item





Für eine Ungeordnete Liste:


- First item

- Second item

- Third item



Um Code zu schreiben:



```
$ echo "hallo wie geht es dir"
```


Die Trennlinie macht man folgendermassen: 
---


Einen Link fügt man so ein:
[title](https://www.example.com)



Ein Bild folgendermassen:

![alt text](image.jpg)

` Formatierung als Code `

Inline Code:
`git status`

- [x] Erledigter Task
- [ ] Unerledigter Task
